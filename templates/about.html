Luminote
Avengers-inspired note-taking app for heroes and developers
Project Overview

Luminote is a modern, Avengers-inspired note-taking web application built with FastAPI, MongoDB, and Bootstrap. It features a bold, responsive UI, secure backend, and a seamless user experience for managing notes. The project demonstrates clean coding, API optimization, and a focus on usability and design.

Features
  • Avengers-themed, responsive UI
  • Add, view, and mark notes as important
  • MongoDB backend for scalable data storage
  • FastAPI for high-performance APIs
  • Bootstrap for modern, mobile-friendly layouts
  • Clean code, security, and API best practices
Tech Stack
  • Python (FastAPI)
  • MongoDB
  • Bootstrap 5
  • Jinja2 Templates
